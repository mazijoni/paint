# React Paint App

En paint-app laget med React.

## Del 1 - Planlegging

- Brukte 8 fredager på å finne en ide til prosjektet
- Endte opp med en Paint-app fordi det virket kult å kombinere tegning og koding

## Hva den kan gjøre

- **Brush** - Tegn med forskjellige farger
- **Eraser** - Viskeler
- **Bucket** - Fyll områder med farge
- **Color Picker** - Velg hvilken farge du vil
- **Clear** - Tøm alt

## Hva jeg brukte

- React (useState, useRef, useEffect)
- HTML5 Canvas
- CSS
